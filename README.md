# MeshCore Responder

Automatischer Antwort-Bot für MeshCore-Direktnachrichten.

## Was macht das Programm?

- Empfängt Direktnachrichten (DMs) über ein MeshCore-Radio (z.B. Heltec V3)
- Antwortet automatisch mit technischen Infos (z.B. RSSI, SNR, Hops, Pfad)
- Dokumentiert alle Aktivitäten in CSV-Dateien (Excel/Numbers-kompatibel)

## Hardware

- Raspberry Pi (Zero 2 W oder Pi 4B)
- MeshCore-Radio (Heltec V3) via USB-Serial
- Frequenz: 868 MHz EU/Narrow/Switzerland
